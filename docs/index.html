Font profiles

Typography is part of the generated UI.

The skill can include a font family, CSS/download URL, fallback stack, and reason. Use confirmed brand fonts when available; otherwise choose a safe free Korean UI font.

Brief field example 
font_profile:
  family: "Pretendard"
  css_url: "https://cdn.jsdelivr.net/.../pretendard.css"
  fallback: "-apple-system, BlinkMacSystemFont, Apple SD Gothic Neo, Noto Sans KR, sans-serif"
  confidence: "recommended fallback"
  reason: "Neutral Korean mobile UI with strong numeric readability"

What the skill generates

Designed for clear handoff to AI design and code agents.

01

Screen brief

Goal, user intent, layout structure, component hierarchy, and interaction notes.

02

JSON UI spec

Machine-readable screen type, sections, components, states, copy, and tokens.

03

Language mode

Choose English or Korean output. The skill should not mix both languages unless asked.

04

Font profile

Recommended font, CSS URL, fallback stack, and license-check reminder.

05

Implementation handoff

Prompt text for coding agents, Figma agents, or design-system implementation.

06

Mobile constraints

Safe areas, touch targets, fixed CTAs, keyboard behavior, and accessibility.
