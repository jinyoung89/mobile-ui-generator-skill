What the skill generates

Generates the UX/UI decisions behind the visible screen.

01

Pattern system

Pattern groups, layout archetype, component inventory, interaction model, and state matrix.

02

Screen brief

Goal, user intent, hierarchy, sections, fixed/sticky regions, and copy direction.

03

Language mode

Choose English or Korean output. The skill should not mix both languages unless asked.

04

Font profile

Recommended font, CSS URL, fallback stack, and license-check reminder.

05

Design handoff

Clear notes for turning the generated UI into production without losing UX intent.

06

Mobile constraints

Safe areas, touch targets, fixed CTAs, keyboard behavior, and accessibility.
