에이전트 스킬 · 모바일 UX/UI 생성기

모바일 앱 요청을 실제 UI/UX 화면으로 바꿉니다.

필요한 화면을 요청하면 스킬이 생성될 모바일 UI의 위계, 컴포넌트, 상태, 카피, 타이포그래피, 모바일 제약까지 잡아줍니다.

요청한 화면

자연어로 필요한 모바일 앱 화면을 요청합니다.

생성 UI

모바일 화면 레이아웃, 시각적 위계, 표시 컴포넌트, CTA 배치.

상태 설계

로딩, 오류, 빈 상태, 권한, 복구, 파괴적 액션 상태.

시각 시스템

타이포그래피, 간격, 톤, 접근성, 모바일 플랫폼 제약.

실제 스킬 산출물 예시

어떤 요청을 했을 때 어떤 UI/UX 결과가 나오는지 보여줍니다.

예시는 코드가 아니라 사용자가 보게 될 결과 화면에 집중합니다. 요청 프롬프트, 생성된 모바일 화면, 시각적 위계, 인터랙션 상태, UX 결정을 함께 보여줍니다.

생성 산출물

한국 핀테크 휴대폰 본인인증

JavaScript가 프롬프트에서 생성된 UI 결과 예시를 렌더링합니다.

폰트 프로필

타이포그래피는 생성 화면의 인상을 바꿉니다.

스킬은 화면에 맞는 한국어 UI 폰트, 대체 톤, 선택 이유를 함께 제안합니다. 확인되지 않은 독점 브랜드 폰트는 만들지 않습니다.

스킬이 생성하는 것

보이는 화면을 만드는 UX/UI 결정을 생성합니다.

01

패턴 시스템

패턴 그룹, 레이아웃 아키타입, 컴포넌트 목록, 인터랙션 모델, 상태 매트릭스.

02

화면 브리프

목표, 사용자 의도, 위계, 섹션, 고정/스티키 영역, 카피 방향.

03

언어 모드

영어 또는 한국어를 선택합니다. 요청하지 않으면 두 언어를 섞지 않습니다.

04

폰트 프로필

추천 폰트, CSS URL, fallback stack, 라이선스 확인 메모.

05

디자인 전달

생성 UI의 UX 의도가 구현 과정에서 사라지지 않도록 전달 메모를 정리.

06

모바일 제약

안전 영역, 터치 타깃, 고정 CTA, 키보드 동작, 접근성.
