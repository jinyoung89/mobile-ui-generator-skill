폰트 프로필

폰트도 생성되는 UI의 일부입니다.

스킬은 폰트명, CSS/다운로드 URL, 대체 폰트 스택, 사용 이유를 함께 제안합니다. 확인된 브랜드 폰트가 있으면 우선 사용하고, 없으면 무료 한국어 UI 폰트를 안전하게 선택합니다.

스킬이 생성하는 것

AI 디자인/코드 에이전트에 명확하게 전달되도록 설계했습니다.

01

화면 브리프

목표, 사용자 의도, 레이아웃 구조, 컴포넌트 위계, 인터랙션 메모.

02

JSON UI 스펙

화면 유형, 섹션, 컴포넌트, 상태, 카피, 토큰을 기계가 읽을 수 있게 정리.

03

언어 모드

영어 또는 한국어를 선택합니다. 요청하지 않으면 두 언어를 섞지 않습니다.

04

폰트 프로필

추천 폰트, CSS URL, fallback stack, 라이선스 확인 메모.

05

구현 전달문

코딩 에이전트, Figma 에이전트, 디자인 시스템 구현용 프롬프트.

06

모바일 제약

안전 영역, 터치 타깃, 고정 CTA, 키보드 동작, 접근성.
